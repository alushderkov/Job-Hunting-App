Задание для лабораторной работы №2 по дисциплине  
«Разработка программного обеспечения для мобильных
платформ»

Требования:

1. Архитектура: разделение бизнес-логики и уровня представления
   (MVVM, MVP или др).
2. Веб-API: получение данных из внешнего сервиса (NewsAPI,
   OpenWeatherMap и др)
3. Мониторинг: проверка наличия интернет-соединения.
4. Оффлайн-режим: кэширование данных из API (использовать локальную
   базу данных или кэш-библиотеки).
