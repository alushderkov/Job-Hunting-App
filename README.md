Задание для лабораторной работы №1 по дисциплине  
«Разработка программного обеспечения для мобильных
платформ»

Цель лабораторных работ: разработать мобильное приложение с
поэтапным внедрением функционала.
Технология разработки на выбор: React Native + Expo.  
Тема приложения: Платформа для поиска удалённой работы

Требования:
1. Пользовательский интерфейс: минимум 3 страницы (например,
   основной экран, экран просмотра деталей, экран настроек).
2. Экран загрузки: отображение сплеш-экрана при запуске приложения.
3. Поддержка локализации: минимум два языка (например, русский и
   английский).
4. Тема интерфейса: выбор светлой или темной темы с сохранением после
   перезагрузки.
5. Локальная база данных: создание, удаление и редактирование записей с
   минимум тремя полями (например, заголовок, описание, дата).
